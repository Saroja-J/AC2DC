# ⚡ AC to DC Converter

A simple **AC to DC Converter** designed in **KiCad** using a bridge rectifier, filter capacitor, and LED indicator. The circuit converts AC voltage from a transformer secondary into a smooth DC output suitable for low-power electronic applications.

---

## ⚙️ How It Works

1. AC voltage is supplied from the transformer secondary.
2. Four **1N4007** diodes form a full-wave bridge rectifier.
3. The filter capacitor smooths the rectified voltage.
4. The LED indicates that power is available.
5. The DC output is provided through the output connector.

---

## ✨ Features

✔️ Simple and beginner-friendly design

✔️ Full-wave bridge rectification

✔️ Reduced ripple using filter capacitor

✔️ LED power indication

✔️ Designed using KiCad

---

## 🛠️ Components Used

- 4 × 1N4007 Diodes
- 1000 µF Capacitor
- 10 kΩ Resistor
- 2.2 kΩ Resistor
- LED
- Input & Output Connectors
